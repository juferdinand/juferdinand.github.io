ByteBuddy

ByteBuddy ist ein einzelner Discord Bot welchen ich für mich und meine Freunde entwickelt habe.

Einladungslink

Projektübersicht

Ich habe diesen Bot schnell entwickelt, um spezifische Funktionen zu integrieren, die in anderen Bots fehlten, aber von unserer Gruppe benötigt wurden. Dazu gehören das Extrahieren von Spielen aus einem Store, um sie unserer Spiele-Wunschliste hinzuzufügen, ein Voting-System zur Auswahl des nächsten Spiels und einige kleinere Features. Als Hinweis: Der Discord Bot wurde so entwickelt, dass manche Funktionen nur auf einem Server funktionieren, da wir den Bot ausschließlich auf unserem Server verwenden. Das hat zum Beispiel damit zu tun dass die Datengrundlage nicht pro Server getrennt ist.
